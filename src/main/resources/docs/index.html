Editor Documentation

About

A plain text editor focused towards and optimized for handling source codes. It was created as a school project.

Features

File handling
Files are controlled either via main menu items or buttons located on one of two available toolbars. All buttons are situated in the top left side of the application window. You can perform basic actions, such as creating, opening or saving files.
Menu
Menu consists of four sections, which are File, Edit, Properties and Help. Section File contains actions mentioned in filehandling, Edit allows you to further modify your document, Properties handle all available settings and Help provides additional information about the application.
Back / forward
This feature allows you to recover recent changes to your document and also redo all changes that you've recovered. Limit of edits that you can walk through is set to 1500. The feature is accessible from both menu and toolbar.
Search
Search utility allows you to scan current document and highlight the first set of characters that fits your search. Repetitive pushing of the search button will cause the application to iterate though the document, highlighting next convenient strings / words.
Settings + config file
Settings are accessible via the Properties submenu, and allow you to modify various application parameters, such as automatic line wparring, language or window dimensions.

Hotkeys

Available hotkeys
Create new file <CTRL> + N
Open a file <CTRL> + O
Save the file <CTRL> + S
Save the file as <CTRL> + <SHIFT> + S
Save all files <CTRL> + <SHIFT> + <ALT> + S
Search the document <CTRL> + F
Undo <CTRL> + Z
Redo <CTRL> + <SHIFT> + Z
Close current tab <CTRL> + Q
