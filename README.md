# Editor

A Java Swing-based text editor.

## Build

```
mvn clean package
```

## Run

```
java -jar target/editor-1.0.0.jar
```

## Project Structure

- Source code: `src/main/java/io/schupke/editor/`
- Resources: `src/main/resources/`
- Main class: `io.schupke.editor.Editor`
